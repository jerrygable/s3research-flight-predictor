Flight Predictor is used to predict the flight path of a High Altitude Balloon (HAB) flight.

NOTE:  The prediction is only as good as the input data.  Make sure Burst Altitude, Ascent and Descent Rates are as accurate as possible.

Quick Start Instructions

  1. Enter the Prediction Info
    1. Select the "Prediction Info" tab
    2. Enter the data required to make a prediction
    3. Click "Generate Prediction". 
  2. View the Prediction Results
    1. Select "Prediction Results" to view the prediction map.
    2. Click the Auto Fit Map to zoom to the markers on the map.
    3. You can drag the border between the Table, Map, and Altitude graph to change their sizes.
Repeat steps 1 and 2 to add more predictions to the results.

As the prediction is generated, the wind data is downloaded as blocks from NOAA.  Each prediction time is downloaded separately.  Each block takes 15 to 40 seconds to download so long flights will take some time to generate.  The data is saved so generating additional scenarios will be fast .  There is a limit to the amount of data kept so the early data may be deleted if you do a very long prediction.

The launch and touchdown altitudes are retrieved from google maps.  You can also put in your own altitudes if you want.  For an in-flight floater prediction you can just use an altitude slightly below the float altitude.

You can enter a float time for floaters.  The burst altitude becomes the float altitude.

You can run many predictions by just making changes to the Prediction info and re-running.  All will be displayed on the map.

You can turn on and off the waypoints, markers and lines for each prediction by clicking on the check boxes in the table on the results page.

You can turn on a 2 marker ruler to measure distances.

Checking the show wind box will show the weather grids downloaded on the map.

You can load predictions you have saved as well as tracks from actual flights saved as CSV files from aprs.fi or findu.com.

And finally, you can save a prediction to the s3research flight that can be viewed on aprs.fi along with the actual flight.  You can find saved overlays here: http://www.s3research.com/tools/overlays
